# Tipos de Licencias de Software

| Tipo de Licencia                         | Características                                                                                                                   |
| ---------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **MIT License**                          | Permite reutilización con pocas restricciones. Se puede usar en software propietario siempre que se incluya la licencia original. |
| **Apache License 2.0**                   | Permite el uso comercial, modificación y distribución, pero requiere la inclusión de la licencia y notas de cambios.              |
| **GPL (General Public License)**         | Obliga a que cualquier derivado sea también de código abierto y mantenga la misma licencia.                                       |
| **LGPL (Lesser General Public License)** | Similar a GPL, pero permite el uso en software propietario si se enlaza de manera dinámica.                                       |
| **BSD License**                          | Muy permisiva; permite redistribución, uso comercial y modificaciones con pocas restricciones.                                    |
| **Creative Commons (CC)**                | No es específicamente para software, pero se utiliza para documentos, imágenes y otros contenidos digitales.                      |
| **Mozilla Public License 2.0**           | Código modificado debe mantenerse bajo la misma licencia, pero permite combinar con software propietario.                         |
| **Eclipse Public License**               | Similar a MPL, pero con mayor enfoque en colaboración y contribución abierta.                                                     |
| **Artistic License 2.0**                 | Principalmente usada para Perl, permite modificaciones y redistribución con ciertas condiciones.                                  |
| **Unlicense**                            | Coloca el software en el dominio público, permitiendo el uso sin restricciones.                                                   |
